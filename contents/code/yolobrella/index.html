Instructions

  • That undulating blob following your cursor around is a baby cloud.
  • Change the size of your cloud using the up and down arrow keys.
  • When your cloud gets big enough, it'll start to rain.
  • If you don't let it get too wet or dry, a tiny little plant might start to grow.
  • If you take really good care of your plant, by giving it even amounts of shade and sunshine...
  • teensy-weensy people might start living off the plant. loljk, there aren't any people.
